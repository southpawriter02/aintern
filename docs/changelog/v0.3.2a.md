## Features

### Observable Properties
Name, Path, ItemType, IsExpanded, IsSelected, IsLoading, IsRenaming, EditingName, IsModified, IsVisible, IsHighlighted, HasChildren, Depth

### Computed Properties
IsDirectory, IsFile, Extension, Language, IconKey, ChildrenLoaded, ShowExpander, IndentMargin, RelativePath

### Lazy Loading
- `LoadChildrenAsync()` loads on first expansion
- `RefreshChildrenAsync()` forces reload
- `InvalidateChildren()` clears cache

### Inline Rename
- `BeginRename()` / `CommitRenameAsync()` / `CancelRename()`
- Validates empty, unchanged, invalid characters

### Filtering
- `ApplyFilter()` recursive with auto-expand
- `MatchesFilter()` case-insensitive
- `ClearFilter()` resets visibility

### Icon Resolution
- Special folders: src, test, docs, node_modules, .git
- 50+ file extension mappings
- Special files: README, package.json, .gitignore
