# v0.5.2f - Main Window Integration

## Summary
Integrated terminal panel into main window with keyboard shortcuts, GridSplitter, and status bar button.

## Added
- **MultiplyConverter** - Dynamic max height calculation (70% of window)
- **Terminal keyboard shortcuts**:
  - `Ctrl+`` - Toggle terminal panel
  - `Ctrl+Shift+`` - New session
  - `Ctrl+Shift+W` - Close active session
  - `Ctrl+PageDown/PageUp` - Tab navigation
- **Status bar terminal button** - Quick toggle with icon

## Modified

### MainWindow.axaml
- Changed Grid from 2-row to 4-row layout (Content, Splitter, Terminal, StatusBar)
- Added GridSplitter for resizable terminal height
- Added TerminalPanel with height constraints (100px min, 70% max)
- Added 5 keyboard shortcuts for terminal operations
- Added terminal toggle button to status bar

### MainWindow.axaml.cs
- Added terminal panel initialization with `ITerminalService`
- Added using statements for DI and terminal interfaces

### MainWindowViewModel.cs
- Added `TerminalPanelViewModel` property (singleton)
- Added constructor parameter for terminal panel

### ServiceCollectionExtensions.cs
- Registered `TerminalPanelViewModel` as singleton
