# v0.5.3b – Shell Configuration System

> Shell-specific commands, OSC escape sequences, syntax elements, and integration scripts.

## Overview

This version creates a comprehensive shell configuration system providing shell-specific behavior for all supported shells.

## Changes

### Added

- **`ShellConfiguration`** model with:
  - Shell-specific commands (clear, cd, pwd, history, ls)
  - OSC escape sequence support flags (OSC 7, 9, 133)
  - Syntax elements (separators, continuations, env vars)
  - Profile file paths and login/interactive arguments
  - Default environment variables (TERM, COLORTERM)

- **`IShellConfigurationService`** interface with:
  - `GetConfiguration(ShellType)` / `GetConfiguration(string)`
  - `FormatChangeDirectoryCommand(type, path)`
  - `GetCwdReportingEscapeSequence(type, dir)`
  - `GetDefaultEnvironment(type)`
  - `GenerateShellIntegrationScript(type)`

- **`ShellConfigurationService`** implementation with:
  - Pre-built configurations for 12 shell types
  - Path escaping per shell type (quotes, spaces, special chars)
  - OSC 7/9 CWD reporting sequence generation
  - Shell integration script generation (Bash, Zsh, Fish, PowerShell)
