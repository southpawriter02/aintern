# v0.3.5f: Global Drag-Drop

**Date:** 2026-01-15

## Summary

Drag-drop file/folder support on MainWindow. Folders open as workspaces, files open in editor.

## Changes

### MainWindow.axaml
- Added drop overlay Border inside Grid with RowSpan="2"

### MainWindow.axaml.cs
- Added `InitializeDragDrop()` - registers event handlers
- Added `OnDragEnter()` - shows overlay for file drops
- Added `OnDragLeave()` - hides overlay
- Added `OnDragOver()` - updates message and drag effect
- Added `OnDrop()` - processes folders/files

### FileExplorerViewModel.cs
- Added `OpenFolderByPath()` - opens workspace by path
- Added `RequestOpenFile()` - raises FileOpenRequested event

### Icons.axaml
- Added `DropIcon` StreamGeometry

## Drop Behavior

| Dropped Item | Action |
|--------------|--------|
| Folder | Open as workspace |
| File(s) | Open in editor tab(s) |
| Mixed | Workspace + files |

## Overlay Messages

| Content | Message |
|---------|---------|
| Folder | "Open as workspace" |
| Single file | "Open filename.ext" |
| Multiple files | "Open N files" |
