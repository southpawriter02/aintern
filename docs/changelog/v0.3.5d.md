## Features
- Workspace segment: folder icon + name, click to open
- File segment: active file name, click to reveal
- Language segment: display name, click to change
- Cursor segment: Ln/Col/selection, click for Go To Line
- Encoding/LineEnding segments with tooltips
- Unsaved indicator (yellow dot + count)
- File watcher sync icon with opacity indicator
- Model status with loading progress
