## Summary
Enhances the status bar with dedicated service/ViewModel architecture for pending changes indicator, model status, save state, and temperature display.

## New Features

### Status Bar Service
- `IStatusBarService` interface and `StatusBarService` implementation
- Aggregates status from `IUndoManager`, `ILlmService`, `ISettingsService`
- Thread-safe `ConcurrentDictionary` for item management
- Event-driven updates with `ItemChanged`, `ConfigurationChanged`

### Status Bar Models
- `StatusBarSection` enum (Left, Center, Right, FarRight)
- `StatusColor` enum (Default, Success, Warning, Error, Info, Muted)
- `StatusBarItem` record with factory methods
- `StatusBarConfiguration` record for show/hide toggles

### ViewModel Enhancements
- Extended `StatusBarViewModel` with pending changes count and temperature display
- `StatusBarItemViewModel` for individual item display
- Navigation events: `ShowChangeHistoryRequested`, `ShowTemperatureSliderRequested`

### Styling
- `StatusBarStyles.axaml` with item, badge, and indicator styles
- `StatusColorConverter` for enum-to-brush mapping
