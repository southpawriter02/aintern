# v0.4.4f Changelog - Batch Preview

**Released**: 2026-01-16

## Summary

Added modal dialog for previewing all file changes in a multi-file proposal before applying.

## Features

### Dialog (`src/AIntern.Desktop/Views`)
- **`BatchPreviewDialog.axaml`** - Modal window with:
  - Header with summary stats (total/new/modified files, +/- lines)
  - Horizontal scrollable file tabs with icons and badges
  - Diff viewer content area
  - Footer with navigation and action buttons
- **`BatchPreviewDialog.axaml.cs`** - Keyboard navigation (Left/Right, Home/End, 1-9)

### ViewModels (`src/AIntern.Desktop/ViewModels`)
- **`BatchPreviewDialogViewModel`** - Navigation commands, file selection
- **`DiffPreviewViewModel`** - Individual file wrapper for diff display

### Converters (`src/AIntern.Desktop/Converters`)
- **`ReferenceEqualsConverter`** - Object reference comparison for selection

## Tests

- 18 new tests (12 for DiffPreviewViewModel, 6 for ReferenceEqualsConverter)

## Dependencies

- v0.4.2: DiffViewerViewModel, DiffResult, DiffStats
- v0.4.4e: FileTreeProposalPanel (integrates preview command)
