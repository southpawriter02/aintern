# v0.4.3g: Conflict Dialog

**Release Date:** January 16, 2026  
**Type:** Feature (Apply Changes Workflow)  
**Parent:** v0.4.3 Apply Changes Workflow

---

## Overview

v0.4.3g introduces the Conflict Warning dialog for user resolution when file conflicts are detected.

---

## New Files

| File | Purpose |
|------|---------|
| `Models/ConflictResolution.cs` | Resolution enum (Cancel/RefreshDiff/ForceApply) |
| `ViewModels/ConflictWarningDialogViewModel.cs` | MVVM ViewModel |
| `Views/ConflictWarningDialog.axaml` | Dialog layout |
| `Views/ConflictWarningDialog.axaml.cs` | Code-behind with ShowAsync |

---

## ConflictResolution Enum

| Value | Description |
|-------|-------------|
| `Cancel` | Abort operation |
| `RefreshDiff` | Recompute diff against current content |
| `ForceApply` | Overwrite despite conflict |

---

## Dialog Features

- **Warning Header**: Icon and title
- **Conflict Details**: Message, explanation, file info card
- **Timestamp Comparison**: Last modified vs snapshot time
- **Resolution Buttons**: Refresh Diff (recommended), Force Apply (warning), Cancel
- **Keyboard Shortcuts**: Enter→Refresh, Escape→Cancel
