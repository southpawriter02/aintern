> User-configurable shell profiles with appearance, behavior, and metadata.

## Overview

This version creates the data models for user-configurable shell profiles.

## Changes

### Added

- **Enumerations:**
  - `TerminalCursorStyle` (Block, Underline, Bar)
  - `ProfileCloseOnExit` (Always, OnCleanExit, Never)
  - `TerminalBellStyle` (Audible, Visual, Both, None)
  - `DirectorySyncMode` (ActiveTerminalOnly, AllLinkedTerminals, Manual)

- **`ShellProfileDefaults`** class with application-wide default values

- **`ShellProfile`** model with:
  - Identity: Id, Name, IconPath, AccentColor
  - Shell config: ShellPath, ShellType, Arguments, StartingDirectory, StartupCommand, Environment
  - Appearance (nullable): FontFamily, FontSize, ThemeName, CursorStyle, CursorBlink, ScrollbackLines
  - Behavior: CloseOnExit, BellStyle, TabTitleFormat
  - Metadata: IsDefault, IsHidden, IsBuiltIn, SortOrder, CreatedAt, ModifiedAt
  - Methods: Clone(), IsValid property

### Changed

- Extended `AppSettings` with 17 terminal settings:
  - Font: TerminalFontFamily, TerminalFontSize
  - Cursor: TerminalCursorStyle, TerminalCursorBlink
  - Behavior: TerminalBellStyle, TerminalScrollbackLines
  - Panel: TerminalPanelHeight, ShowTerminalOnStartup
  - Selection: TerminalCopyOnSelect, TerminalWordSeparators
  - Shortcuts: TerminalToggleShortcut, TerminalNewShortcut
  - Sync: SyncTerminalWithWorkspace, TerminalSyncMode
  - Profile: DefaultShellProfileId, RestoreTerminalSessions
