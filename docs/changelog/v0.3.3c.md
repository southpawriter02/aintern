## API Reference

| Method | Description |
|--------|-------------|
| `ApplyHighlighting(editor, language)` | Applies syntax highlighting to an editor |
| `SetLanguage(editor, language)` | Changes grammar without reinstallation |
| `ChangeTheme(theme)` | Updates theme for all registered editors |
| `RemoveHighlighting(editor)` | Removes and disposes installation |
| `GetScopeForLanguage(language)` | Returns TextMate scope for language |
| `IsLanguageSupported(language)` | Checks if language has grammar |
