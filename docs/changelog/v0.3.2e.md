## API

```csharp
// XAML MultiBinding usage
<MultiBinding Converter="{x:Static converters:IconColorConverter.Instance}">
    <Binding Path="IconKey" />
    <Binding Path="IsDirectory" />
</MultiBinding>

// Programmatic usage
Color color = IconColorConverter.GetColorForIconKey("file-csharp");
IReadOnlyDictionary<string, Color> all = IconColorConverter.GetAllColors();
```
