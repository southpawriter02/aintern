# v0.4.4e Changelog - Proposal Panel

**Released**: 2026-01-16

## Summary

Added Avalonia UI components for displaying interactive multi-file proposals with tree view, selection controls, and apply workflow.

## Features

### Controls
- **`FileTreeProposalPanel`** - Main panel with header, tree, options, and actions
  - Header with icon, title, description, and stats
  - Error/warning banners for validation feedback
  - Selection controls (Select All, Deselect All, Expand/Collapse)
  - Progress display during apply
  - Keyboard shortcuts (Ctrl+A, Ctrl+D, Ctrl+P, Ctrl+Enter, Esc)
- **`FileTreeItemControl`** - Individual tree items with checkboxes, icons, badges

### Converters
- **`IconNameConverter`** - Maps icon names to StreamGeometry resources
- **`SelectionStateToCheckStateConverter`** - Tri-state checkbox binding
- **`ValidationSeverityToBrushConverter`** - Severity to color brush

### Styles
- **`ProposalPanelStyles.axaml`** - Icons (StreamGeometry) and color brushes

## Tests

- 235 tests pass across Desktop.Tests

## Dependencies

- v0.4.4d: Tree ViewModels (FileTreeProposalViewModel, FileOperationItemViewModel)
