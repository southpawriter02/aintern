# v0.4.4g Changelog - Progress Overlay

**Released**: 2026-01-16

## Summary

Visual overlay during batch apply operations with phase feedback, progress bar, and cancellation.

## Features

### Control (`src/AIntern.Desktop/Controls`)
- **`ApplyProgressOverlay.axaml`** - Semi-transparent backdrop with centered modal card
- **`ApplyProgressOverlay.axaml.cs`** - Escape key handling for cancellation

### ViewModel (`src/AIntern.Desktop/ViewModels`)
- **`ApplyProgressViewModel`** - Visibility, phase, progress, cancellation, auto-hide (1500ms)

### Converters (`src/AIntern.Desktop/Converters`)
- **`PhaseToIconConverter`** - Maps phase to StreamGeometry icons
- **`PhaseToTitleConverter`** - Maps phase to display titles
- **`PhaseToBrushConverter`** - Maps phase to foreground colors (success/error/accent)

## Tests

- 24 new tests (10 for ViewModel, 14 for converters)

## Dependencies

- v0.4.3: `BatchApplyPhase`, `BatchApplyProgress` models
